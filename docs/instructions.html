📖 Инструкция по использованию мониторинга

🌊 Что такое OkaFloodMonitor?

Автоматическая система мониторинга паводка на реке Оке в районе Серпухова. Работает 24/7, данные обновляются 4 раза в день (08:00, 12:00, 17:00, 20:00 МСК).

В версии v7.0 добавлен источник GloFAS Flood API — глобальная система прогнозирования стока рек, дающая данные по 7 станциям верховий Оки и 16-дневный прогноз.

📊 Откуда берутся данные?

Уровень воды в Серпухове — сайт администрации г. Серпухова (serpuhov.ru), пост наблюдения д. Лукьяново на р. Ока. Публикуется ежедневно в ~09:00.

Другие станции (Кашира, Калуга, Рязань) — API навигационного сервиса КИМ. Абсолютные уровни в см от нуля соответствующего поста (не сравниваются напрямую с Серпуховом).

Суточные приросты и прогнозы — Центральное УГМС (ЦУГМС), ежедневные обзоры развития весеннего половодья.

Погода и снежный покров — Open-Meteo API (бесплатный).

GloFAS (верховья Оки) — GloFAS Flood API. Расход воды (м³/с) для 7 станций: Орёл, Мценск, Белёв, Козельск, Калуга, Алексин, Таруса. Прогноз 16 дней. Без API-ключа, бесплатно.

🔢 Как читать данные?

Уровень воды (см)

Высота воды над нулём водомерного поста д. Лукьяново. Нуль поста ≈ 107.54 м БС. Формула: абсолютный = 107.54 + уровень_м

< 400 см — НОРМА. Паводковый сезон в начальной фазе.
400–600 см — ВНИМАНИЕ. Пойма начинает заполняться.
600–800 см — ОПАСНОСТЬ. Пойма затоплена, приближаемся к НЯ.
> 800 см — КРИТИЧНО. Немедленно следите за сводками.

НЯ и ОЯ

НЯ (113.99 м БС, ≈645 см) — неблагоприятное явление — уровень выхода воды на пойму.

ОЯ (115.54 м БС, ≈800 см) — опасное явление — уровень подтопления населённых пунктов.

Расход GloFAS (м³/с)

Объём воды, протекающей через поперечное сечение русла в секунду. Flood ratio (×N) — отношение текущего расхода к среднему: >3 = высокий паводок, >5 = экстремальный.

🌡 Паводковый индекс погоды

Комплексный показатель риска подъёма воды (0–4), вычисляется по данным Open-Meteo:

  • 0 — СТАБИЛЬНЫЙ: морозы сдерживают таяние
  • 1 — УМЕРЕННЫЙ: незначительное таяние
  • 2 — ПОВЫШЕННЫЙ: снег тает, осадки умеренные
  • 3 — ВЫСОКИЙ: значительный риск подъёма
  • 4 — КРИТИЧЕСКИЙ: активное таяние + осадки + Rain-on-Snow

❓ Часто задаваемые вопросы

Почему нет других источников данных?
fishingsib.ru и allrivers.info прекратили публикацию данных (последние данные — 2024 год). КИМ API в Серпухове мёртв с ноября 2022. Основной источник — serpuhov.ru.

Что такое «Жерновка»?
Локальное название местности примерно в 8 км ниже Пущино по реке Ока. Паводковая волна от Серпухова доходит туда примерно за 6–12 часов.

Как работает GloFAS Timeline?
Горизонтальная шкала показывает прогнозируемое время пика на каждой станции и расчётное время прибытия волны в Серпухов, исходя из типичной скорости волны добегания.
