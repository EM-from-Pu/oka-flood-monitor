🌊 Данные о паводке на Оке

  • serpuhov.ru — Паводковая обстановка Серпухова — ежедневные данные о уровне воды у д. Лукьяново ok
  • ЦУГМС — Весеннее половодье 2026 — ежедневные обзоры развития паводка
  • ЦУГМС — Уточнённый прогноз пика для бассейна верхней Оки
  • КИМ — уровни воды по гидропостам — API для Каширы, Калуги, Рязани
  • GloFAS Flood API (Open-Meteo) — расход воды на 7 станциях верховий Оки, прогноз 16 дней

🌦 Погода

  • Open-Meteo — погода и снежный покров (бесплатный API, используется в мониторинге)
  • Meteoinfo.ru — прогноз погоды ФГБУ ЦПГиМ
  • rp5.ru — подробный прогноз для Серпухова
